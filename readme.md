# GYC WordPress Theme


## Bash commands
```bash
# Install dependences
npm install

# Watch for Gutenberg blocks updates
npm run start:blocks

# Watch for CSS tailwindcss theme changes
npm run start:theme

# Watch for js theme changes (frontend)
npm run start:js

# Build Gutenberg blocks
npm run build:blocks

# Build CSS theme
npm run build:theme

# Build JS theme assets
npm run build:js
```
